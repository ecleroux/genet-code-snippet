# Change Log

All notable changes to the Genet JSON Formatter extension will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.1] - 2025-11-05

### Enhanced
- Exclude hidden files and folders (any path segment starting with a dot) from snippet list
- Improved snippet file collection logic for better accuracy

## [1.0.0] - 2025-11-04

### Added
- Initial release of Genet: Code Snippet VS Code extension
- Multi-folder snippet support
- Live preview of snippet content
- Keyboard navigation and accessibility improvements
- Pagination for large snippet sets
- Strict TypeScript implementation
- QuickPick UI for snippet selection
